# Mountain Car Project

A compact project exploring the classic **Mountain Car** reinforcement
learning task.

## Structure

-   **book_images/** -- reference images\
-   **generated_images/** -- visuals from training\
-   **notebooks/** -- experiments and plots\
-   **src/** -- core implementation

## Summary

The agent must learn to escape a valley by building momentum, since the
car cannot climb the hill directly.

## Implementation

Training setups, results, and code are located in the notebooks and
source folder.
